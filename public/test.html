JSON Pretty Print

When developing php applications, it is often helpful to dump variables. var_dump() or print_r() can used, but are not very readable for large complex data structures. The dBug() class is a great alternative which improves readability, but is bad at distinguishing numbers, strings, null values, and boolean values. JSON Pretty Print (jpp()) seeks to address these issues by displaying data as colour-coded and formatted JSON. Let's dive in...

Initialize you data

<?php
    require 'class.JSON.php';
    $json = new JSON($data);
?>

The class can inject your css

<?php
    echo $json->css();// default theme
    //echo $json->css('emacs');// or use an alternate theme
?>

Which will output something like this

}

    color: black;
    text-transform: uppercase;
    background-color: rgba(224, 109, 151, 0.75);
    padding: 0 .6em;

}

code.json .string {
    color: whitesmoke;
}
</style>

Or if you prefer, just do this

<link rel="stylesheet" href="theme.default.css" />

Now, output your data

<?php
    echo $json->prettyprint();
?>

Which will give you something that looks like this

The short-hand for all the above is:

<?= jpp($x); ?>
